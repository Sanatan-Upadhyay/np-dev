Week-5 Discuss

Q1
In which of the following scenarios is amortized complexity likely to be a more accurate measure of algorithmic efficiency ?
a)

Worst case cost of each operation is independent of the other operations.

b)

Worst case cost of an operation depends on the preceding sequence of operations.

c)

Average cost of an operation across random inputs can be computed.

d)

Inputs are constrained to come in a sequence that reduces the overall worst case complexity.

Answer

b

Discuss
Q2
In the Union-Find data structure, suppose we want to add an operation reassign( j, k ) that moves item j to the same component as item k. What would be the complexity of this operation for a set with n elements ?
a)

O(n) for both the array representation and the pointer representation.

b)

O(1) for the array representation and O(log n) for the pointer representation.

c)

O(n) for the array representation and O(log n) for the pointer representation.

d)

O(1) for both the array representation and the pointer representation.

Answer

d

Q3
In a min-heap, what is the most accurate description of the worst-case complexity of the operation find_max that reports the value of the largest element in the heap, without removing it ?
a)

O(n log n)

b)

O(n)

c)

O(log n)

d)

O(1)

Answer

b

Q4
After inserting 57 into a max-heap, the structure of the heap is [ 82, 57, 27, 42, 25, 18, 25, 27, 32 ]. What was the structure of the heap before inserting 57 ?
a)

[ 82, 42, 32, 27, 25, 18, 25, 27 ]

b)

[ 82, 27, 42, 32, 25, 18, 25, 27 ]

c)

[ 82, 42, 27, 25, 32, 18, 25, 27 ]

d)

[ 82, 42, 27, 32, 25, 18, 25, 27 ]

Answer

d

Q5
Consider an alternative to binary search called ternary search that divides the input array into three equal parts and recursively searches one of these three segments. What can we say about the asymptotic worst case complexity of ternary search versus binary search ?
a)

The complexity of ternary search is strictly better than that of binary search.

b)

The complexity of ternary search is the same as that of binary search.

c)

The complexity of binary search is strictly better than that of ternary search.

d)

The relative complexity of ternary and binary search depends on the distribution of values across the array.

Answer

b

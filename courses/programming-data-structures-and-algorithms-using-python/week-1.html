Q2
What is h(28) - h(27), given the definition of h below ?

def h(n):
   s = 0
   for i in range(1,n):
     if n%i == 0;
        s = s + i
   return(s)
a)

(Type: int) 15

b)

(Type: double) 15

c)

(Type: String) 15

d)

None Of The Above

Answer

c     ** h(n) adds up the factors of n from 1 to n-1. h(28) is 28 and h(27) is 13, so the difference is 15.

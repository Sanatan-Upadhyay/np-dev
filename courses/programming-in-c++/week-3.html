Q1
Identify the necessary correction required in data member/s in the given program below to get the output Area = 20.

#include < iostream >
using namespace std;

class rectangle {
  int w, h;
public:
  rectangle(int a, int b) : w(new int(a)), h(new int(b)) {}
  ~rectangle() { delete w, h; }
  int area() { return(*w * *h); }
};

int main() {
  rectangle p(4, 5);
  cout << "Area = " << p.area();
  return 0;
}

The data member/s should be of
a)

Reference type

b)

Pointer type

c)

constant type

d)

Derived type

Answer

b

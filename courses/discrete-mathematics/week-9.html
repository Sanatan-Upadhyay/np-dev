a)

( 1 + x + x2 + x3 + ... ) ( 1 + x2 + x4 + x6 + ... )

b)

( 1 + x + x2 + x3 + ... ) ( 1 + x2 + x4 + x6 + ... )2

c)

( 1 + x + x2 + x3 + ... )2 ( 1 + x2 + x4 + x6 + ... )

d)

( 1 + x + x2 + x3 + ... )3 ( 1 + x2 + x4 + x6 + ... )

c

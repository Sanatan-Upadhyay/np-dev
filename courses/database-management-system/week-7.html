a)

If Timestamp(Ti) ≤ Write-timestamp(Q), the read operation is executed

b)

If Timestamp(Ti) ≤ Write-timestamp(Q), Read-timestamp(Q) is set to max(Read-timestamp(Q), Timestamp(Ti))

Q1
Given the following permutation of a, b, c, d, e, f, g, h, i, j. What is the previous permutation in lexicographic (dictionary) order? Write your answer without any blank spaces between letters.

fjadcbeghi
a)

(Type: Regex Match) ]*fjadbihgec[]

b)

(Type: Regex Match) ]*\'fjadbihgec\'[]

c)

(Type: Regex Match) ]*\"fjadbihgec\"[]

d)

All Of The Above

Answer

d     ** Invert the algorithm given in the video. Look for the longest suffix increasing order, here beghi. The letter before is c. Replace by the next smallest letter in teh suffix, b and arrange the remaining letters in descending order, to get bihgec.

Q3
Suppose we add this function foo() to the class Tree that implements search trees. For a name mytree with a value of type Tree, what would mytree.foo() compute?

def foo(self):
  if self.isempty():
    return(0)
  elif self.isleaf():
    retrun(self.value)
  else:
    return(self.value + max(self.left.foo(), self.right.foo()))
a)

The sum of the elements in the tree

b)

The length of the longest root to leaf path in the tree

c)

The maximum sum across all root to leaf paths in the tree

d)

The number of root to leaf paths in the tree

Answer

c     ** This computes the maximum sum along the paths from the root to the leaves

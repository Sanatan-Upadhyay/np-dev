Q1
One of the following 10 statement generates an error. Which one

x = [1,"abcd",2,"efgh",[3,4]]     # Statement 1
y = x[0:50]     # Statement 2
z = y     # Statement 3
w = x     # Statement 4
x[1] = x[1][0:3] + 'd'     # Statement 5
y[2] = 4     # Statement 6
z[0] = 0     # Statement 7
x[1][1:2] = 'yzw'     # Statement 8
w[4][0] = 1000     # Statement 9
a = (x[4][1] == 4)     # Statement 10
a)

(Type: int) 8

b)

(Type: String) 8

c)

(Type: double) 8

d)

None Of The Above

Answer

b     ** x[1] is a string, which is immutable, so assignment to the slice x[1][1:2] is not permitted.

Q2
Consider the following lines of Phython code

x = ['a', 42, 'b', 377]
w = x[1:]
y = x
u = w
w = w[0:]
w[0] = 53
x[1] = 47

Which of the following is correct ?
a)

x[1] == 47, y[1] == 47, w[0] == 53, u[0] == 42

b)

x[1] == 47, y[1] == 47, w[0] == 53, u[0] == 53

c)

x[1] == 47, y[1] == 42, w[0] == 53, u[0] == 53

d)

x[1] == 47, y[1] == 42, w[0] == 53, u[0] == 53

Answer

a     x and y are aliased. w and u are not. So the update to x[1] affects y[1] while the update to w[0] leaves u[0] unchanged

Q3
What is the value of second after executing the following lines ?

first = "wombat"
second = ""
for i in range(len( first ), 0, -1):
  second = first[i-1] + second
a)

(Type: Regex Match) \s*\"wombat\"\s*

b)

(Type: Regex Match) \s*\'wombat\'\s*

c)

All of the above

d)

None Of The Above

Answer

c     ** The loop copies the string from right to left

Q4
What is the value of list1 after the following lines are executed ?

def mystery(l):
    l = l[2:5]
    return()

list1 = [7, 82, 44, 23, 11]
mystery(list1)
a)

(Type: Regex Match) \s*\[\s*7\s*,\s*82\s*,\s*44\s*,\s*23\s*,\s*11\s*]\s*

b)

(Type: Regex Match) \b*\[\b*7\b*,\b*82\b*,\b*44\b*,\b*23\b*,\b*11\b*]\b*

c)

(Type: Regex Match) \d*\[\d*7\d*,\d*82\d*,\d*44\d*,\d*23\d*,\d*11\d*]\d*

d)

None of the above

Answer

a     Since l is reassigned a value within the function, it refers to a new value and the update does not affect the value of list1 outside the function
